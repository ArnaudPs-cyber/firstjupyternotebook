As a cloud data analyst at TheLook Fintech, I have successfully built the dashboard Trevor and their team need to monitor 
the status of loans.
First, I created a visualization to display the total amount of outstanding loans.
Second, I created a visualization to display the total amount of outstanding loans by status.
Third, I created a visualization to display the top 10 states with the highest total count of outstanding loans.
Fourth, I created a visualization that displays customers who own their home outright and have “Current” loans - as filters 
on the dashboard -
Finally, I enabled cross-filtering on the loan status dashboard to make it easier for the users to interact with the data 
and understand how one measure relates to others and explored how to enable automatic refreshes to visualization.
I am well on my way to understanding how to use Looker to build dashboards that help businesses monitor and track 
key performance indicators (KPI).
